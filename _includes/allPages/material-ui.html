Breaking Changes
  • [core] Upgrade react-tap-event-plugin and react peer dependency (#6057) @rosskevin

We have upgraded the peer dependency of Material-UI. You gonna need:

  • react-tap-event-plugin >= 2.0.1
  • react >= 15.4.0
  • react-dom >= 15.4.0
