Release Notes on 2018 M07 10 for 6.3.3

Highlights

  • In the next minor version we will remove the beta versions of Angular 6 as valid dependency versions. This is a change that we intended to make when moving out of beta but was accidentally overlooked. Please note that versions prior to beta.8 don't work properly even today, as they do not have all of the changes we depend on related to tree-shakeable providers.
  • We have added a cdk-experimental package for drag & drop. It's not ready for prime time yet, but if you're feeling adventurous and want to give it a spin, feel free!

Bug Fixes
